# Architectural Portfolio & Project Management Platform

A premium, production-grade architectural portfolio built with React, TypeScript, and Firebase. This application features a sophisticated design, advanced project discovery tools, AI-powered insights, and a robust admin dashboard for project management.

## ✨ Key Features

### 🏛️ Portfolio & Discovery
- **Cinematic Visuals**: High-resolution project hero sections with 21:9 aspect ratios and editorial-style typography.
- **Advanced Filtering**: Discover projects by category (Residential, Commercial, Cultural, Industrial), custom tags, and completion year.
- **Timeline Range**: Filter projects within a specific "From" and "To" year range.
- **Global Search**: Real-time keyword search across project titles, descriptions, and locations.
- **Interactive Maps**: Dynamic Google Maps integration for each project location with premium hover effects.
- **AI Design Insights**: Automated project summaries generated by Google's Gemini AI, highlighting design philosophy and structural achievements.

### 🛠️ Admin & Management
- **Full CRUD Dashboard**: Add, edit, and remove projects through a secure administrative interface.
- **Automated Slug Generation**: SEO-friendly URLs are automatically generated from project titles.
- **SEO Optimization**: Custom fields for Meta Titles and Meta Descriptions to enhance search engine visibility.
- **Rich Content Support**: Manage multi-paragraph descriptions, image galleries, and external project resources.

### 🚀 Performance & UX
- **Fluid Animations**: Smooth transitions and scroll-triggered animations powered by Framer Motion and GSAP.
- **PWA Ready**: Installable as a Progressive Web App for a native-like experience on mobile and desktop.
- **Responsive Design**: Desktop-first precision with mobile-first code, ensuring a seamless experience across all devices.
- **Real-time Updates**: Powered by Firebase Firestore for instantaneous data synchronization.

## 🛠️ Tech Stack

- **Frontend**: React 18+, Vite, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (`motion/react`), GSAP
- **Backend**: Firebase (Firestore, Authentication)
- **AI Integration**: Google GenAI (Gemini 3 Flash)
- **Data Fetching**: TanStack Query (React Query)
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- A Google Cloud project with Firebase and Gemini API enabled.

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Create a `.env` file based on `.env.example`.
   - Add your `GEMINI_API_KEY` and Firebase configuration.

### Development
Run the development server:
```bash
npm run dev
```

### Production Build
Build the application for production:
```bash
npm run build
```

## 📂 Project Structure

- `src/screens`: Main page components (Home, Projects, Detail, Admin).
- `src/components`: Reusable UI components (ProjectCard, AiDisclaimerModal, etc.).
- `src/hooks`: Custom React hooks for Firebase data fetching and state management.
- `src/data`: Initial seed data and TypeScript interfaces.
- `src/firebase.ts`: Firebase SDK initialization and configuration.
- `firebase-blueprint.json`: Intermediate representation of the Firestore data structure.
- `firestore.rules`: Security rules for protecting project data.

## 🔒 Security

The application uses Firebase Security Rules to ensure that:
- Project data is publicly readable but only writable by authenticated administrators.
- Data integrity is maintained through strict schema validation within the rules.
